Instruções - Dashboard de Análises Pontuais

Dashboard Correto

Para acessar o dashboard de análises pontuais que mostra apenas as linhas de produção cadastradas para o turno atual, use o link correto:

Dashboard Pontual (Linhas de Produção)
Dashboard Antigo

URL: /qc/dashboard/spot/legacy/

Problema: Mostra todos os locais do sistema, não apenas os cadastrados para o turno atual.

Características:

  • Mostra "Módulo I" e "Módulo II"
  • Exibe todos os locais do sistema
  • Não filtra por produção ativa
Dashboard Novo (Correto)

URL: /qc/dashboard/spot/

Vantagem: Mostra apenas linhas de produção cadastradas para o turno atual.

Características:

  • Filtra por produção ativa do turno
  • Mostra apenas linhas cadastradas
  • Exibe apenas produtos cadastrados
  • Agrupa dados por linha de produção
Como Funciona o Dashboard Correto
1. Busca Produção Ativa

Localiza a produção cadastrada para o turno atual

2. Filtra Linhas

Mostra apenas linhas de produção cadastradas

3. Agrupa Dados

Organiza informações por linha de produção

Se Não Há Dados

Se o dashboard estiver vazio, significa que não há produção cadastrada para o turno atual. Neste caso:

  1. Clique no botão "Cadastrar Produção"
  2. Preencha os dados da produção
  3. Cadastre as linhas de produção ativas
  4. Cadastre os produtos que serão produzidos
  5. Volte ao dashboard para ver os dados
